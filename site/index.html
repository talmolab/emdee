emdee

A fast, minimal, cross-platform markdown viewer.
Opens .md files natively on macOS, Windows, and Linux.

curl -fsSL https://emdee.tlab.sh/install.sh | sh
irm https://emdee.tlab.sh/install.ps1 | iex

or download from GitHub Releases · view source

Features

GFM

GitHub-Flavored Markdown

Tables, task lists, strikethrough, footnotes, and autolinks — everything you expect.

Syntax Highlighting

25+ languages via Prism.js — just annotate your fenced code blocks.

KaTeX Math

Inline $...$ and block $$...$$ LaTeX rendering.

Mermaid Diagrams

Flowcharts, sequence diagrams, and more. Lazy-loaded — zero cost when not used.

Dark / Light Theme

Toggle between dark and light modes. Follows your OS preference by default.

Table of Contents

Auto-generated sidebar from headings with scroll tracking.

In-Document Search

Ctrl/Cmd+F with match highlighting and navigation between results.

Multi-Window & File Associations

Each file opens in its own window. Registers as handler for .md, .markdown, .mdown, .mkd, .mdx.
